# tenrrr
# 🚀 Tenrrr – DSA Practice Platform

A simple, fast, and focused **DSA practice tracker + question hub** built using pure HTML.

🔗 Live Site: https://tenrrr.vercel.app

---

## 📌 What This Project Does

Tenrrr is designed to help you:

* Practice **DSA topic-wise**
* Track your **problem-solving progress**
* Stay consistent without distractions
* Access structured problem sets quickly

No login. No complexity. Just solve.

---

## 🧠 Features

* 📚 Topic-based DSA questions
* 📄 Multiple question pages (Part 1, 2, 3…)
* ⚡ Lightweight (pure HTML, no frameworks)
* 🌐 Deployed on Vercel
* 🧭 Easy navigation between problem sets

---

## 📁 Project Structure

```
tenrrr/
│── index.html                     # Main landing page
│── tennr-questions-part2.html     # Question set (Part 2)
│── tennr-questions-part3.html     # Question set (Part 3)
│── code.txt                      # Additional notes / code
│── README.md
```

## 🚀 How to Run Locally

```bash
git clone https://github.com/manishrao0312/tenrrr.git
cd tenrrr
open index.html
```

Or just double-click `index.html`.
